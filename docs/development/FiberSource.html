Class: FiberSource

FiberSource

FiberSource A fiber bundle in Phantomas is defined as a cylindrical tube wrapped around its centerline. The centerline itself is a continuous curve in 3D, and can be simply created from a few control points. All the fibers created are supposed to connect two cortical areas. FiberSource is the basic Class for the representation of a Fiber. Objects containing the geometries to be added to the scene are to be referred to FiberSource for gathering any necessary information.
